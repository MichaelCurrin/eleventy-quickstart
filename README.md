# Eleventy Quickstart
> Starter template for a static site built with Eleventy

Learn more in my [Eleventy resources](https://michaelcurrin.github.io/dev-resources/resources/javascript/packages/eleventy/).


## Installation

```sh
$ npm install
```


## Usage

Dev server.

```sh
$ npm start
```

Then open the browser at:

- http://localhost:8080

Build for a deploy.

```sh
$ npm run build
```
